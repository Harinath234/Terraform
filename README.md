These projects showcase how to automate infrastructure provisioning using **Terraform**, including multi-region deployments, remote backends, modules, and production-ready 3-tier architectures.

## 🚀 How to Run Terraform

Initialize Terraform:

```
terraform init
```

Review infrastructure plan:

```
terraform plan
```

Deploy infrastructure:

```
terraform apply
```

---

## 🧹 Destroy Infrastructure

To remove all created resources:

```
terraform destroy
```
